# ApiRequest

This function is used to make API calls. It takes a `request` object and a `cb` callback function as input. The `request` object contains the HTTP method (GET, POST or PUT), headers and body of the request. The `cb` function is called with the response data.

### Usage

```js
const { ApiRequest } = require("@starberry/gatsby-theme-utils/APIRequest")

let requestObj = {
  method: "", // request method
  header: "",
  body: "",
} // request object

let cb // callback function to handle response

let apiResponse = ApiRequest(requestObj, cb)
// ...
```

### Parameters

| Name         | Type       | Description                                   |
| :----------- | :--------- | :-------------------------------------------- |
| `requestObj` | `object`   | The request object.                           |
| `cb`         | `function` | The callback function to handle the response. |

### Returns

| Type     | Description          |
| :------- | :------------------- |
| `object` | The response object. |

### Examples

Here's an example of using the `APIRequest` utility to make a `GET` request:

```js
import { ApiRequest } from "@starberry/gatsby-theme-utils/APIRequest"

const requestObj = {
  method: "GET",
  header: {
    Authorization: "Bearer myToken",
  },
  body: {
    param1: "value1",
    param2: "value2",
  },
} // request object

const cb = (response) => {
  console.log(response)
}

const apiResponse = ApiRequest(requestObj, cb)
```

And here's an example of using the APIRequest utility to make a POST request:

```js
import { ApiRequest } from "@starberry/gatsby-theme-utils/APIRequest"

const requestObj = {
  method: "POST",
  header: {
    "Content-Type": "application/json",
    Authorization: "Bearer myToken",
  },
  body: {
    param1: "value1",
    param2: "value2",
  },
} // request object

const cb = (response) => {
  console.log(response)
}

const apiResponse = ApiRequest(requestObj, cb)
```
