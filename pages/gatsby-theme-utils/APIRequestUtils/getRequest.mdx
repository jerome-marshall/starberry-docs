# getRequest

The getRequest function makes a `GET` request to the specified URL with optional request headers and parameters. It takes a `request` object as input, which should contain the URL, headers, and body of the request. You can also provide a callback function to handle the response.

### Usage

```js
const { getRequest } = require("@starberry/gatsby-theme-utils/APIRequest")

let requestObj = {
  url: "", // API endpoint
  header: "", // optional request headers
  body: "", // optional request parameters
}
let cb
let apiResponse = getRequest(requestObj, cb)
// ...
```

### Parameters

| Name    | Type     | Description                                                            |
| :------ | :------- | :--------------------------------------------------------------------- |
| request | object   | An object containing the request information.                          |
| url     | string   | The URL to make the request to.                                        |
| header  | object   | An object containing any additional headers to include in the request. |
| body    | object   | An object containing any parameters to include in the request.         |
| cb      | function | A callback function to handle the response.                            |

### Returns

| Type   | Description                                                                  |
| :----- | :--------------------------------------------------------------------------- |
| object | An object containing the response status, headers, and body of the response. |

### Example

```js
const { getRequest } = require("@starberry/gatsby-theme-utils/APIRequest")

let requestObj = {
  url: "https://example.com/api/posts",
  header: {
    Authorization: "Bearer TOKEN",
  },
  body: {
    userId: 1,
    limit: 10,
  },
}

const handleResponse = (data) => {
  console.log(data)
}

getRequest(requestObj, handleResponse)
```
